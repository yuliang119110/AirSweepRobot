# 🚁 倾斜旋翼八旋翼飞行器 — 仿真与模型资料包

> 一个 8 旋翼飞行器（无人机）的**控制算法仿真**和**3D 打印模型**开源资料包。
> 它的特别之处在于：8 个旋翼都向外倾斜了 30°，能更稳地抵抗外界干扰，适合做"飞行 + 操作物体"这类研究。

---

## 一、这是什么？（用大白话讲）

普通的无人机，螺旋桨都是**朝上**吹风的，要让飞机往侧面移动，得先把整个机身"歪"过去。
而这个项目里的飞机，8 个螺旋桨是**斜着**安装的（向外倾斜 30°），所以它能**不用歪机身**就直接向侧面出力气，
抗风吹、抗碰撞的能力更强，也更适合一边飞一边去推、拉、抓东西。

这个项目目前放进文件夹里的是两样东西：
- **一套仿真程序**：用电脑模拟这个飞机"悬停 + 被随机干扰"时，控制器怎么稳住它。
- **一套 3D 模型**：机架的 CAD 文件（`.STL` / `.STEP`），可以直接拿去 3D 打印或加工。

## 三、这个飞机是怎么被"稳住"的？

程序里用了一套叫 **Sample-based MPC（基于采样的模型预测控制）** 的方法。通俗地说，控制器的思考过程是这样的：

1. 我现在在哪、状态如何？
2. 我**同时**在脑子里"彩排"很多种（默认 512 种）未来几秒的螺旋桨推力方案；
3. 每种方案都拿动力学公式"预演"一遍，看看哪种最能让飞机稳在目标点（且最省力气）；
4. 选最好的那一种，只执行它的**第一步**，然后下一刻重新来过。

因为要做成百上千次并行预演，程序用了 **JAX**（一个能调动 GPU/CPU 并行计算的库）来加速。
飞机质量约 **1.0 kg**，8 个旋翼最大推力各约 **9.81 N**，目标悬停高度 **1 米**。

## 四、快速开始（跑一次仿真）

### 1. 准备环境（需要 Python 3.8+）

```bash
pip install jax jaxlib numpy matplotlib
```

### 2. 运行仿真

```bash
python tilted_octorotor_mpc_simple.py
```

程序会：
- 在终端打印每一步的位置、速度、计算耗时；
- 在文件夹里生成（或覆盖）上面那 6 张 `.png` 仿真图。

### 3. 想改参数？

打开 `tilted_octorotor_mpc_simple.py`，找到最下方的 `main()` 函数，可以改：

```python
config = OctorotorConfig(
    n_samples=512,          # 采样数：越大越准但越慢（可试 2048）
    sim_duration=20.0,      # 仿真时长（秒）
    disturbance_force_max=5.0,   # 最大扰动力（牛）
    disturbance_torque_max=0.1,  # 最大扰动扭矩（牛·米）
)
```

## 五、6 张仿真图怎么看？

| 图 | 横轴 | 看什么 |
|---|---|---|
| `01` 位置 / 速度 | 时间 | Z 高度应该贴着 1 米的目标线；X/Y 应该贴近 0 |
| `02` 3D 轨迹 | — | 飞机从起点（绿）飞到终点（红），目标点（金星）在 (0,0,1) |
| `03` 推力分配 | 时间 | 8 条线围绕"悬停推力"上下波动；总推力应≈飞机重量(9.81N) |
| `04` 扰动 | 时间 | 每隔约 2 秒随机换一次外力/力矩，看控制器怎么扛 |
| `05` 位置误差 | 时间 | 离目标点的距离，越低越好（图里标了平均/最大误差） |
| `06` XY 俯视 | — | 水平方向漂移大不大，越聚在原点附近越好 |

## 六、3D 模型文件怎么用？

- **`.STL`**：最常见的 3D 打印格式。用 Cura / PrusaSlicer 等切片软件打开，即可打印机架配件。
  本包含 `airframe1/2/3` 三个不同的机架部件。
- **`.STEP`**：通用 CAD 格式，可用 SolidWorks、Fusion 360、FreeCAD 等打开和编辑整机模型 `Flerobot.STEP`。
